# Functional Test: QUERY Workflow

## Objetivo
Verificar que QUERY responde preguntas sobre un proceso basándose en spec.md existente.

## Pre-condiciones
- spec.md existente para un proceso
- Skill instalado

## Pasos
1. Iniciar con: "Explicame cómo funciona el proceso de aprobación de compras"
2. Verificar que lee spec.md del proceso correcto
3. Verificar que responde en español con información de la spec
4. Hacer pregunta de follow-up: "¿Quiénes son los actores?"
5. Verificar respuesta basada en sección de Actores de la spec

## Resultado esperado
- Respuestas en lenguaje natural basadas en la spec
- No modifica archivos (read-only)
